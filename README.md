# 🚜 EasyFinder AI

**EasyFinder AI** is a production-ready backend platform designed to identify, score, and prioritize heavy equipment listings across multiple marketplaces using configurable, data-driven logic.

It is built to support buyers, dealers, and enterprise procurement teams who need **actionable intelligence**, not raw listings.

---

## 🌐 Live Environment

- **Production API:** https://easyfinder.fly.dev  
- **Health Check:** https://easyfinder.fly.dev/api/health  
- **Infrastructure:** Fly.io (Machines)
- **Runtime:** Node.js 20
- **Database:** MongoDB

---

## 🧠 Core Capabilities

- Centralized equipment listings API
- Scoring & ranking engine (price, hours, state, operability)
- Configurable scoring weights
- Preferred-state biasing
- MongoDB-backed persistence
- Production deployment with zero-downtime restarts
- Health monitoring & diagnostics
