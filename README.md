# EasyFinder AI

EasyFinder is an AI-powered lead scoring and prioritization platform that helps teams identify high-intent prospects instantly.

## Why It Matters
Sales teams waste time chasing low-quality leads. EasyFinder surfaces the best opportunities in seconds.

## Architecture
- Frontend: React + Vite
- Backend: FastAPI
- Auth: JWT
- Infra: Fly.io + Docker

## Security
- JWT authentication
- Role-based access control
- Rate-limited APIs
- No PII logged

## Status
Production-ready MVP
